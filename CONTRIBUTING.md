# Contributing

## Running Tests

```bash
./mvnw test
```

## Running Integration Tests

```bash
./mvnw verify
```

## Code Style

This project follows the Quarkus code style. Format your code before opening a PR:

```bash
./mvnw process-sources
```

## Opening Issues

Please check [Quarkus Issue #53104](https://github.com/quarkusio/quarkus/issues/53104) for context.
